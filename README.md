# tourism-recommendation
사용자 성향에 따른 관광지역 추천 프로젝트
(2022.05.11 ~ 2022.06.21)

## 🔨 핵심 기능
1. 여행지 데이터를 크롤링하여 특성을 추출한 뒤 사용자 성향에 따라 추천 
2. 관광지역 추천 후 기상지수 기준으로 정렬 

## 💻 활용 SW
- Beautiful Soup 
- 기상청_단기예보 ((구)_동네예보) 조회서비스 (https://www.data.go.kr/data/15084084/openapi.do)
- Apache Mahout 라이브러리
