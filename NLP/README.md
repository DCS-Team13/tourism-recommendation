- 분석 품질 (띄어쓰기유무, 오탈자에도 어느 정도 품질 요구)

# KOMORAN vs mecab
||KOMORAN|mecab|
|---|---|---|
|연산 속도|로딩시간: 0.9542 / 형태소 분석 시간: mecab에 비해 느림(다른 라이브러리에 비해서는 빠른 편)|로딩시간: 0.0004 / 형태소 분석 시간: 압도적으로 빠름|
|분석 품질|오탈자가 있고 띄어쓰기가 없는 문장에서도 분석 품질 좋음|KOMORAN에 비해 떨어지는 품질, 그래도 품질 나쁘지 않은 편|
|언어|자바|파이썬|

> ### 이유
